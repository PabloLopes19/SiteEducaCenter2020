Desenvolvimento de sistemas

O que é:

Esse curso é voltado para aquela pessoa que se interessa em programação e desenvolvimento de aplicações que irão facilitar e ajudar pessoas a fazerem o seu trabalho. Você vai aprender a criar uma interface em janela, armazenar dados do servidor e organiza-los com algoritmos. Além de C#, você vai aprender sobre visual basic, SQL, e a integração do BackEnd com o FrontEnd da aplicação!

O que você vai aprender:

O curso de Programação C# com Visual Studio foi idealizado, sendo este o terceiro de quatro módulos voltados para automação comercial. Com um conteúdo desenvolvido com base nos conceitos fundamentais de programação orientada a objetos e as funcionalidades da ferramenta, este módulo apresenta novos conceitos para o aluno além de aprofundar alguns conceitos abordados no módulo anterior.
